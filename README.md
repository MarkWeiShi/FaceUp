# 顔鉴 · AI 东亚颜值分析 H5

> 专为东亚审美校准的 AI 颜值分析工具 —— 不只给分，更给出路。

## 📱 产品特点

- **东亚美学体系**：基于中日韩越四国审美标准的 8 维度评分系统
- **0-9 分制评分**：符合东亚用户语言习惯的评分机制
- **三层改善建议**：护肤 × 彩妆 × 医美 完整改善链路
- **多地区模式**：支持中国、韩国、日本、越南四种审美权重切换
- **社交分享**：3 套精美分享卡片模板

## 🚀 快速开始

### 本地运行

1. 克隆或下载本项目
2. 使用本地服务器打开项目目录（需要 ES Module 支持）

```bash
# 方式一：使用 Python
python -m http.server 8080

# 方式二：使用 Node.js（需安装 serve）
npx serve .

# 方式三：使用 VS Code Live Server 插件
```

3. 在浏览器中打开 `http://localhost:8080`

### API Key 配置

本项目使用 Anthropic Claude API 进行 AI 分析。首次使用时需要配置 API Key：

1. 访问 [Anthropic Console](https://console.anthropic.com/) 获取 API Key
2. 在应用中点击设置按钮或首次分析时输入 API Key
3. API Key 仅存储在本地 sessionStorage，不会上传至服务器

## 🔧 技术栈

- **前端框架**：原生 HTML + CSS + Vanilla JS（无框架依赖）
- **AI 分析**：Anthropic Claude API (claude-sonnet-4-6)
- **图表**：原生 Canvas 2D API
- **分享卡片**：html2canvas
- **字体**：Google Fonts
